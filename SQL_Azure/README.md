This repository will document my experimentation with Microsoft Azure and SQL Databases.

*Setup Azure*
- Access https://azure.microsoft.com/en-ca/ and sign in/ sign up for the Azure Trial 

*Setup Database*
- It is good practice to set up a DB inside a resource
- Click All Resources
- Select Add
- Select SQL Database
- Create the new resource group, db name, server, etc and voila

*Access Database*
- Click All Resources
- Click Server
- Collect Server ID
- Go to VSC
- New File
- Language to SQL
- Command Pallet
- MS:SQL Connect
- Input Server ID that you copied
- Fill in the rest of the information like DB Name, Username, Password
- Connect
- Try a query
